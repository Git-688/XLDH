# ⭐️ XLDH 星链导航

一款极简、高效的现代个人导航页。  
一站直达主流网站、工具和音乐，支持自定义内容，体验流畅且美观。

## ✨ 特性

- 三级导航结构，查找资源更快更清晰
- 智能搜索、多引擎切换
- 当日壁纸与简洁问候
- 一键畅听主流音乐榜单
- 天气、节日等实用小组件
- 响应式设计，桌面与移动端完美适配

## 🚀 快速开始

1. 克隆仓库  
   `git clone https://github.com/Git-688/XLDH.git`
2. 静态托管源码即可访问（如 GitHub Pages、Vercel等）
3. 可自定义 `/data/navigation.json` 与 `/assets/` 资源

## 📂 目录概览

```
index.html      # 首页入口
/assets/        # 图片与图标
/css/           # 样式
/js/            # 脚本
/data/          # 导航数据
/pages/         # 扩展页面
```
